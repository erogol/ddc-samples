Mozilla TTS - Double Decoder Consistency Samples.

DDC samples

The purpose of this page is to show the strength of the DDC in order to train reliable TTS models in a very short time with a robust attention alignment and faster than real-time inference performance.

Model Details: These samples are generated using DDC model and MultiBand-MelGAN vocoder. DDC was trained 90k steps using a single GPU for 2 days as explained in the blog post. The vocoder was trained using real spectrograms for 1 million steps. These two models provides speech synthesis with ~0.1 real-time factor on a GPU and ~0.35 on a CPU.
